# Java-Fundamentals

## Overview

This repository focuses on the core concepts of Java programming. It contains beginner-friendly exercises designed to build a strong understanding of Java syntax and programming fundamentals.

## Topics Covered

* Variables
* Data Types
* Methods
* Loops
* Conditional Statements
* Classes
* Objects
* Basic Java Programs

## Technologies

* Java

## Purpose

The goal of this repository is to develop a solid understanding of Java fundamentals before progressing to more advanced programming concepts.
